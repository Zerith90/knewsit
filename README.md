
## Usage

Install dependencies
```
$ npm install
```
or
```
yarn
```

For development
```bash
$ npm run dev
```

Build web app scripts and styles:
```bash
$ npm run build
```

For production
```bash
$ npm start
```


News Source : ![News API](https://newsapi.org/)

- Pulls news from curated sources of data
- Auto load news when scrolling
